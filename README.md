## Introduction
This is a Python script designed for educational and testing purposes only. It encrypts files in a specified directory using the Fernet encryption algorithm and generates a ransom note

## Features
- Encrypts files in a specified directory
- Generates a ransom note with instructions for decryption
- Uses obfuscation techniques to evade detection

## Usage
1. Clone the repository to your local machine.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Run the script by executing `python encrypt_files.py`.
4. Enter the full path of the directory you want to encrypt when prompted.
5. The script will encrypt all files in the specified directory and generate a ransom note.
6. Share the encrypted files and ransom note.
